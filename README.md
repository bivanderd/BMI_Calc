# BMI Calculator 🩺

Kalkulator BMI sederhana berbasis Python.
Dibuat sebagai project pertama dalam perjalanan belajar Python untuk Health Tech.

## Fitur
- Input berat badan (kg) dan tinggi badan (cm)
- Menghitung BMI secara otomatis
- Menampilkan kategori: Underweight / Normal / Overweight / Obesity
